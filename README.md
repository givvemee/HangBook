#### 📌 SUMMARY
- **프로젝트 명** Hang-Book (Haning-Out with Books)
- **프로젝트 개요** API 를 이용하여 베스트 셀러 조회, 책 추천, 검색 기능을 구현한 웹 앱
- **기간** 2021.11 ~ 2022. 01 (약 2달)

<br/>

#### 🛠 STACKS
- **사용 기술** 
    - **React**
        - `Styled-Component` 
        - `React-Router-Dom`
    - **Javascript**
<br/>

#### 🪡 KEY POINTS
- `axios` 를 이용한 api 연동
- Dark mode / Light mode 구현
- keyword query 를 이용한 검색 기능 구현
